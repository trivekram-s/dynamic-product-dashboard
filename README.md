# Product Dashboard

## Project Description

This is a React-based Product Dashboard application built using Vite. The application displays a list of products, allows users to filter products based on availability, and uses conditional rendering to show different UI states.

Features:
- Display product cards dynamically
- Show product name, price, and stock status
- Highlight out-of-stock products
- Display a message when no products are available
- Styled using CSS Modules and Material UI
- Includes automated tests using Vitest and React Testing Library

## Installation

Clone the repository:

```bash
git clone https://github.com/Rajesh2723/product-dashboard.git
```

Navigate to the project folder:

```bash
cd product-dashboard
```

Install dependencies:

```bash
npm install
```

---

## Running the Application

Start the development server:

```bash
npm run dev
```

Open the URL displayed in the terminal (usually):

```text
http://localhost:5173
```

---

## Running Tests

Run the test suite:

```bash
npm test
```

or

```bash
npm run test
```

If using Vitest:

```bash
npm run test -- --run
```
